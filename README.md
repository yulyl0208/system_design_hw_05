#  Fitness Tracker

Сервис фитнес-трекинга, реализованный на C++ с использованием **userver framework**.

Аналог MyFitnessPal с поддержкой:
- пользователей
- упражнений
- тренировок
- статистики

---

##  Технологии

- C++17
- userver 
- PostgreSQL 
- Redis 
- Docker 

---

##  Архитектура

Система состоит из 3 основных компонентов:

- **API сервис userver**
- **PostgreSQL** — хранение данных
- **Redis** — кеширование и rate limiting

---

##  Основные возможности

### Пользователи
- создание пользователя
- поиск по логину
- поиск по маске имени

### Упражнения
- создание упражнения
- получение списка упражнений 

### Тренировки
- создание тренировки
- добавление упражнений
- история тренировок

### Статистика
- агрегация тренировок
- количество повторений

---

##  Оптимизации производительности

###  Кеширование 
- список упражнений
- историю тренировок
- статистику

###  Rate limiting

- POST /workouts → 50 req/min
- GET /stats → 30 req/min
- GET /history → 60 req/min

###  База данных
- индексы на user_id
- оптимизация JOIN запросов
- нормализованная схема

---

##  Метрики производительности

- cache hit rate
- p95 / p99 latency
- RPS
- время выполнения SQL запросов
- нагрузка Redis

---

## Запуск проекта

```bash
docker-compose up --build

API: http://localhost:8080

Получить упражнения: GET /exercises

Создать тренировку: POST /workouts?user_id=1

История тренировок: GET /workouts/history?user_id=1

Статистика: GET /workouts/stats?user_id=1
